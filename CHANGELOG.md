# Changelog

All notable changes to Oryn will be documented in this file.

## [0.0.1] - 2024

### Added
- Initial release
- Claim data model with evidence and counter-evidence
- Time decay engine
- Confidence computation engine
- Claim engine for CRUD operations
- Evidence engine for managing proof
- Local-first storage with Isar
- Minimal UI for claim management
- Claim list screen
- Claim detail screen with confidence breakdown
- Create claim screen
- Attach evidence screen
- README documentation
- MIT License
- Claim Object specification

### Philosophy
- No user accounts
- No reputation systems
- No social features
- Confidence is computed, never declared
- All proof decays over time
- Local-first, offline-first design
